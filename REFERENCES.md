# Recommended Resources

Books and documentation for learning PyQt and Qt. None are bundled — find them through your preferred bookseller or library.

## Official Documentation

- [PyQt6 Reference Guide](https://www.riverbankcomputing.com/static/Docs/PyQt6/) — Riverbank Computing
- [Qt for Python (PySide6)](https://doc.qt.io/qtforpython-6/) — Qt Company's official Python bindings
- [Qt 6 C++ Documentation](https://doc.qt.io/qt-6/) — Source of truth for all Qt classes

## Books

- *Beginning PyQt: A Hands-On Approach to GUI Programming* — Joshua Willman (Apress, 2020)
- *Modern PyQt: Create GUI Applications for Project Management, Computer Vision, and Data Analysis* — Joshua Willman (Apress, 2021)
- *Qt5 Python GUI Programming Cookbook* — B.M. Harwani (Packt, 2018)
- *Create GUI Applications with Python & Qt* — Martin Fitzpatrick (self-published, regularly updated)
- *Rapid GUI Programming with Python and Qt* — Mark Summerfield (Prentice Hall, 2007)

## Icon Resources

- [Fugue Icons](https://p.yusukekamiyamane.com/) — 3,500+ free 16×16 icons by Yusuke Kamiyamane
- Qt's built-in QStyle.StandardPixmap icons — included with PyQt6, no download needed
